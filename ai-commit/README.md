# AI Commit Skill

Generate clean, consistent git commit messages using AI.

## Usage

1. Stage changes:
   git add .

2. Ask agent:
   "commit this"
   OR
   "generate commit message"

3. Review → confirm → commit

## Features

- Conventional Commits
- Smart scope detection
- Diff summarization
- Safe confirmation flow

## Tips

- Keep commits small for best results
- Use "commit and push" for extended workflows
